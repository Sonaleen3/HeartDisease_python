# HeartDisease_python

This project performs **Exploratory Data Analysis (EDA)** on a Heart Disease dataset using **Python and Pandas** in a **Jupyter Notebook (VS Code)**.
The analysis explores different health factors such as **age, gender, chest pain type, cholesterol, and blood pressure** to understand patterns related to heart disease.

---

## 🛠 Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook (VS Code)

---

## 📊 Analysis Performed

* Dataset exploration and cleaning
* Checking null values and duplicates
* Statistical summary of data
* Correlation heatmap
* Distribution analysis of age, cholesterol, and blood pressure
* Gender and heart disease comparison

---

## 📚 Learning Outcomes

* Data cleaning using Pandas
* Exploratory Data Analysis (EDA)
* Data visualization using Matplotlib and Seaborn
